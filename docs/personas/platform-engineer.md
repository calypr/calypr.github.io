# Platform Engineer

Platform engineers implement and run the storage, compute, metadata, and access services that support research delivery.

## Typical Priorities

- Operate stable services across cloud and on-prem environments.
- Keep interfaces standards-aligned and observable in production.
- Support teams without exposing unnecessary infrastructure complexity.

## Where CALYPR Helps

- [Manage Data](../solutions/manage-data.md) for secure object registration and controlled access.
- [Manage Compute](../solutions/manage-compute.md) for portable, repeatable workflow execution.
- [Integrate Data](../solutions/integrate-data.md) for metadata validation and connected context.
