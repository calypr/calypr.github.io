# Data Steward

Data stewards own data quality, access boundaries, and metadata consistency so research teams can use sensitive datasets responsibly.

## Typical Priorities

- Govern who can access what data and under which project boundary.
- Keep metadata complete, validated, and usable for discovery.
- Ensure dataset lifecycle changes remain auditable.

## Where CALYPR Helps

- [Manage Data](../solutions/manage-data.md) for governed registration, references, and controlled movement.
- [Integrate Data](../solutions/integrate-data.md) for validated metadata and discoverable context.
