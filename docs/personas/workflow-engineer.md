# Workflow Engineer

Workflow engineers focus on execution paths, input/output conventions, and operational reproducibility for analysis pipelines.

## Typical Priorities

- Standardize task execution across schedulers and runtimes.
- Preserve provenance between governed inputs and workflow outputs.
- Reduce environment-specific glue code in research workflows.

## Where CALYPR Helps

- [Manage Compute](../solutions/manage-compute.md) for portable workflow execution and task orchestration.
- [Manage Data](../solutions/manage-data.md) for governed inputs and durable references.
