# `create`

## Usage

```bash
gripql-cli create <graph> --host <url>
```

- `<graph>`: The name of the graph to be created (required).
- `--host <url>`: The URL of the GripQL server (default is "localhost:8202").

## Example

```bash
gripql-cli create my_new_graph --host myserver.com:8202
```
